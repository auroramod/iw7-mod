# IDA Scripts

This folder contains IDA scripts that were used to reverse engineer Infinite Warfare more effectively. Each script here can be used on your own IW7 IDB to easily find and debug functions.

## Loading a .idc script in IDA

To load a .idc script, go to `File` in the top left, and click `Script File`. From there, you simply want to select the .idc script of choice.

## Scripts
- **iw7 label GSC builtins.idc** - automatically labels all of the engine functions that GSC builtins have
